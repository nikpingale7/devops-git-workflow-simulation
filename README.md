# DevOps Git Workflow Simulation

This project demonstrates a practical Git and GitHub workflow in a DevOps environment.

## Project Overview

The goal of this project is to simulate collaboration across multiple Linux servers using a centralized GitHub repository.

## Architecture

Mumbai Server (Developer)
        |
        v
      GitHub
        ^
        |
Singapore Server (Operations)

## Features Implemented

- Git repository cloning
- Commit and push operations
- Pull operations across servers
- Feature branch creation
- Pull request workflow
- Merge conflict resolution
